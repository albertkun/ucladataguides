Viewing Data¶

  1. In the Data set page, you can view information about the data, such

    as column names, source of the data, etc.

  2. After familiarizing yourself with the data set, click on “View Data”

  3. You can scroll through the data columns or records

Filtering the data set by date¶

  1. Under “Filter” click the initial filter column

  2. Choose “Arrest Date” as the column to filter by

  3. Next click on “is” and change the filter condition to “is

    after”

  4. Choose after “2016”

    |C:\Users\akocha\AppData\Local\Microsoft\Windows\INetCache\Content.Word\2019-06-24 12_03_31-Arrest Data from 2010 to Present \_ Los Angeles - Open Data Portal.png|

Visualizing the data¶

  1. Click on “Visualize”

  2. Choose whether to “sign in” and save your visualization, or continue

    without being able to save. Note you can use:bigdataforjustice@gmail.com| summer2019!

  3. Click on “Select Column” to select a column to visualize.

  4. Select “Descent Code” to start visualizing the data.

  5. You can scroll down and group the data by other values, for example,

    “Descent Code by Sex”

  6. You can switch the visualization by clicking on one of the buttons

    on the right

  7. The green circle shows a recommended visualization

    |C:\Users\akocha\AppData\Local\Microsoft\Windows\INetCache\Content.Word\2019-06-24 12_09_00-\_ Los Angeles - Open Data Portal.png|

Summarizing our Map Points¶

  1. Scroll down to “Point Aggregation”

  2. Select “Region Map”

  3. Choose “Zip Codes”

  4. Zoom back to Los Angeles again, by either searching “Los Angeles” or

    moving the map to there, now you can see our data by Zip Code |C:\Users\akocha\AppData\Local\Microsoft\Windows\INetCache\Content.Word\2019-06-24 13_08_39-\_ Los Angeles - Open Data Portal.png|

Save the Visualization¶

  1. If you have created a Socrata Account, you can save the Visulization

    by clicking at the bottom, “Save Draft”

  2. Give it a name and click save

  3. You can now “Publish” the visualization and share it via html link.

    Note, you cannot save your visualization as a static image or page.
